To compile & run this code, first download the parsercodegen.c and the needed input files into the same directory.
Use "gcc parsercodegen.c" to generate a program file from the .c file.
You can now run the program while passing in your input file with this
command: "./a.out input.txt".
The expected output will be shown directly in the terminal and an elf.txt file will be created.
